pm2bib
======
Convert Pubmed ID to a BibTex entry.

### Dependency ###

* python (>= 3.2)

### Usage ###

    $ ./pm2bib.py --help

    $ ./pm2bib.py 959840
    @article{nowell76,
            volume = "194",
            title = "{The clonal evolution of tumor cell populations}",
            author = "Nowell, P C",
            pages = "23--8",
            year = "1976",
            number = "4260",
            pmid = "959840",
            journal = "{Science}",
    }
